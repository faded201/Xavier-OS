# 🎧 XAVIER OS - QUICK START CARD

## 🚀 FASTEST WAY TO START

### Windows Users:
**Double-click** `START_SYSTEM.bat` in the library folder

**Done.** All 3 services start automatically. Browser opens. You're done.

---

## 📱 PHONE SETUP (30 seconds)

1. **Find your PC IP** (shown in the startup window, usually `192.168.x.x`)
2. **On your phone:** Open any browser
3. **Type URL:** `http://192.168.1.100:3001` (replace with your IP)
4. **Press Enter** - you're connected!
5. **Tap a book** → **"Awaken Book"** → Enjoy!

## 🆘 IF SOMETHING BREAKS

| Problem | Fix |
|---------|-----|
| **"Can't connect"** | Restart `START_SYSTEM.bat` |
| **No audio** | Check phone volume + app permissions |
| **Slow animations** | Close other apps, restart browser |
| **IP address wrong** | Run START_SYSTEM again, check output |
| **Book won't load** | Check internet connection |

---

## 📊 YOUR PC NEEDS

- **Windows 10+** (or Mac/Linux)
- **Node.js** (already installed)
- **Same WiFi** as phone
- **Firewall open** for ports 3001-3003

---

## ⚡ PORT GUIDE

| Service | Port | What It Does |
|---------|------|--------------|
| **React App** | 3001 | Main UI & player |
| **Express API** | 3002 | Backend routing |
| **TTS Engine** | 3003 | Voice generation |

Only port **3001** needs to work on phone. Others are automatic.

If nothing works:

```
2. Close browser

4. Run START_SYSTEM.bat again
```
